Subject: Basic Web Development

What is web development?

Web development is the work involved into designing a website for the internet. It can range from designing simple static webpage based on plaintext to complex Web-based enterprise grade web applications, electronic services, social networks etc.

A web developer may work on designing a webpage, engineer it, make suitable server to host that webpage. He/she may work on server and network security configuration also.

Web development usually refers to writing the markup and codes for a website.

Technologies for web development

HTML

HTML or Hypertext Markup Language is a language to define the structure of a webpage.

CSS

CSS is a stylesheet language used to define the styles of the elements in a webpage. Example: Setting the background of a button to black

Javascript

Javascript is a programming language used to define the logics of a webpage. As an example, defining what would happen when a button is clicked.

Note: These are the most basic technologies for web development. Developers use more technologies such as SCSS(stylesheet language), Typescript etc for making better webpages.

HTML - HyperText Markup Language

As I said earlier, HTML is a language for defining the structure of a webpage. Developers use it to tell browsers to render which elements. As an example, a developer may use html to show some buttons and texts in the browser.

The basics of HTML

HTML follows a tag like syntax. Which means that, to define an element we use tags. HTML starts with the <html> tag and ends with </html> tag. As you can see, we add / before the tag name to close that. Then we add <head> tag and add tags for title of the webpage, linking stylesheets, scripts etc. Then we close that by typing </head>. Now comes the <body> tag where we define the elements which will be visible in the browser. Like buttons, headers, etc. Like before, we close that tag by typing </body>.

Some basic elements in a page

Buttons

The <button> tag is used to add buttons in a webpage. A simple button with a text "Click me" may have a source code like this <button>Click me</button>.

Headers

In html, there are headers with 6 different size. The biggest header have the <h1> tag while the smallest have the <h6> tag. A simple header may have a source code like this <h1>I'm a header</h1>.

Paragraphs

We define paragraphs using the <p> tag. We then write the contents inside those tags.

Attributes

HTML elements has some attributes which is used for certain operations like getting the element from css, setting the location when a hyperlink is pressed etc. Attributes values are typed inside the starting tag. Like following:
<a href="https://google.com">Google Website</a> Attributes like class, id are used to give elements a different identity. Which is convenient when getting those elements from the html body. class attribute can be used with many different elements where id attribute can only be used in one element in html body. Both attributes can be used in the same elements at the same time.

Simple HTML Example

<html>
<head>
    <title>Simple Webpage</title>
</head>
<body>
    <a href="https://google.com">Google Website</a>
    <button class="welcome-button big-button">Button</button>
    <h1>Biggest Header</h1>
    <h2>Smaller Header</h2>
    <h3>More Smaller Header</h3>
    <h4>More More Smaller Header</h4>
    <h5>More More More Smaller Header</h5>
    <h6>Smallest Header</h6>
</body>
</html>

CSS - Cascading Style Sheet

Cascading style sheet is a way we define the style of an element. CSS is added to a webpage using <style> tag and writing css code in it or adding it from another file using the <link> tag like this <link rel="stylesheet" href="[STYLESHEET_FILE_NAME]" />. The syntax for css is like the following: 

[html tag name/tag class name/tag id name] {
    [property]: [value];
}

Common Properties and their jobs

Property Job Example
background Sets the background color/image of a given element. background: white; background: url('image.jpg');
color Sets the text color of a given element. color: black;
height Sets the height of given element height: 100px;
width Sets the width of a given element. width: 100px;
border Sets the border style of a given element. border: 1px solid red;
font-family Sets the font of the text of a given element. font-family: 'Inter';
font-size Sets the size of the text of a given element. font-size: 10px;

Simple CSS Example

button // button element
{
    font-family: 'Inter';
    color: white;
    background: black;
}
.header // header class
{
    height: 100px;
    color: white;
    background: black;
}
#logo // element with id "logo"
{
    background: url('logo.png');
    width: 100px;
}

JS - JavaScript

Javascript is a programming language which is used to make servers, define logics of a webpage etc. It's the main programming language used to define the logics in a webpage. If you know any other programming languages, it won't be very hard to learn Javascript.

Including in HTML

JS code can be included in the html using <script> tag and directly writing the source code inside it. Or, in cleaner way, it can be inluded from .js file using src attribute of that tag.

Using Javascript to show alert when a button is pressed

We can create a javascript function and set it's name as the value as the onClick attribute of an html button. When it's pressed, the function get's called. And then the function calls alert() function with the text in it. 

<html>
<head>
    <title>Button onClick alert</title>
</head>
<body>
    <button onClick="sayHello()">Say Hello</button>
    <script>
        function sayHello() {
            alert("Hello!");
        }
    </script>
</body>
</html>
